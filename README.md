# 📸 **Visor de Imágenes Interactivo**

¡Bienvenido a mi **visor de imágenes interactivo**! 🚀 Este proyecto te permite visualizar imágenes con descripciones y navegar entre ellas fácilmente utilizando flechas y puntos de navegación. La interfaz es simple y fluida, proporcionando una experiencia agradable para el usuario. 

## 🛠️ **Tecnologías utilizadas**

- **HTML** 📄
- **CSS** 🎨
- **JavaScript** 💻
- **Accesibilidad** ♿

## 🔑 **Características**

- **Navegación por flechas**: Navega entre las imágenes con flechas a la izquierda y derecha. ⬅️➡️
- **Puntos de navegación**: Selecciona una imagen directamente desde los puntos de navegación. •
- **Descripciones**: Cada imagen tiene su propia descripción mostrada en la parte inferior. 📜
- **Animación suave**: Transiciones suaves al cambiar de imagen para una experiencia de usuario más agradable. ✨
- **Accesibilidad mejorada**: Uso de atributos `aria` para facilitar la navegación a usuarios con discapacidad. 🔍
